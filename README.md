## Data Analysis/Machine Learning Internship Portfolio

### About the Repository:
This repository consists of different **Data Science and Machine Learning** related projects, all part of my internship at Kaf InFoTech LLP [Hueristic Academy].
It serves as a reflection of what skills I've achieved, and the domains of interest I followed through with across the period of 3 months.

### Tech Stack:
All projects were programmed in Python. Specifically, Python 3.14.2.
Datasets were extracted from Kaggle, which was extremely helpful.
I used libraries like Pandas, NumPy, MatPlotLib, Seaborn, Scikit-Learn, among others.

### Skills and Concepts Explored:
**1. Data Analysis**:
- Data Collection [Through Kaggle, or in-built datasets. User-Defined datasets have been sparingly used]
- Data Checking [Using functions like head(), tail(), info(), etc.]
- Duplicate Detection [Using duplicate()]
- Missing Value Handling [Either deleting null values using dropna(), or filling them using fillna()]
- Data Transformation [Conversion of data like strings to integers. Eg: Currency]
- Exploratory Data Analysis

**2. Statistics**:
- Descriptive Statistics [Using describe() to show information of the dataset at a glance in text form, or visualisation using graphs]
- Probability
